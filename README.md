springfield-showcase
====================

[Springfield](http://u2ware.github.io/springfield) 를 사용한 간단한 예제 입니다.


코드 얻기:
-------------------
Github 저장소 복사:

    $ git clone https://github.com/u2ware/springfield-showcase.git

Github 에 대해 http://help.github.com 에서 도움을 받을 수 있습니다.


서버 실행 하기 :
-------------------	
command line 에서 Maven 으로 실행하기 :

    $ cd springfield-showcase
    $ mvn tomcat7:run

또는 

만약, SpringSource Tool Suite (STS) 와 같은 IDE 를 쓰고 계신다면:

* "SpringSource tc Server Developer Edition" 이나  Tomcat 등 Servlet 2.5 이상 지원하는 Server 에
springfield-showcase 를 "Add" 합니다.

웹 브라우저 열기 :
-------------------	
http://localhost:8080/yourproject/
